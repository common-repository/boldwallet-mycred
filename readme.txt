[Boldwallet](https://bwalletpay.com) payment method for [myCRED](https://wordpress.org/plugins/mycred/).

== Description ==

[Boldwallet](https://bwalletpay.com) is one of the Financial Technology providers in Nigeria. One of the services which Boldwallet provides is payment gateway service. This plugin enables myCRED to use Boldwallet as a payment gateway.

To signup for a Boldwallet account visit their website by clicking [here](https://bwalletpay.com/myaccount)
 
With this Boldwallet myCRED Payment Gateway plugin, you will be able to accept the following payment methods in your shop:

* __Mastercard__
* __Visa__
* __Verve__
* __PayAttitude__


= Note =

This plugin is meant to be used by merchants in Nigeria for Now.
